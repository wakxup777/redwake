## Prerequisites

- Docker (running)
- An LLM API key from any [supported provider](/llm-providers/overview) (OpenAI, Anthropic, Google, etc.)

## Installation

<Tabs>
  <Tab title="curl">
    ```bash
    curl -sSL https://redwake.rf.gd/install | bash
    ```
  </Tab>
  <Tab title="pipx">
    ```bash
    pipx install redwake-agent
    ```
  </Tab>
</Tabs>

## Configuration

Set your LLM provider:

```bash
export REDWAKE_LLM="openai/gpt-5.4"
export LLM_API_KEY="your-api-key"
```

<Tip>
For best results, use `openai/gpt-5.4`, `anthropic/claude-opus-4-6`, or `openai/gpt-5.2`.
</Tip>

## Run Your First Scan

```bash
redwake --target ./your-app
```

<Note>
First run pulls the Docker sandbox image automatically. Results are saved to `redwake_runs/<run-name>`.
</Note>

## Target Types

RedWake accepts multiple target types:

```bash
# Local codebase
redwake --target ./app-directory

# GitHub repository
redwake --target https://github.com/org/repo

# Live web application
redwake --target https://your-app.com

# Multiple targets (white-box testing)
redwake -t https://github.com/org/repo -t https://your-app.com
```
